TITLE: Variables and Data Types

EXPLANATION:

In Python, a variable is used to store data in memory so that we can use it later in the program.

Think of a variable like a labeled box:
name = "Alex"

Here:
- name is the variable name
- "Alex" is the value stored inside it

Python creates variables using the = assignment operator.

Example:

name = "John"
age = 20
price = 99.5

print(name)
print(age)
print(price)

OUTPUT:
John
20
99.5


RULES FOR VARIABLE NAMES:
1. Must start with a letter or underscore (_)
2. Cannot start with a number
3. Cannot contain spaces
4. Use meaningful names
5. Case-sensitive (age and Age are different)

Valid:
student_name
_age
marks1

Invalid:
1name
my name
class


MAIN DATA TYPES IN PYTHON:

1. Integer (int)
Whole numbers

age = 21
count = 100

2. Float (float)
Decimal numbers

price = 45.75
pi = 3.14

3. String (str)
Text inside quotes

name = "Alex"
city = 'Chennai'

4. Boolean (bool)
True or False values

is_student = True
is_logged_in = False


CHECK DATA TYPE USING type()

name = "Alex"

print(type(name))
print(type(age))

OUTPUT:
<class 'str'>
<class 'int'>


MULTIPLE VARIABLES:

REASSIGNING VARIABLES:

DAY: 4
TITLE: Operators in Python

EXPLANATION:

Operators are special symbols used to perform operations on values and variables.

Example:

a = 10
b = 5

Python uses operators to calculate, compare, and make decisions.


==================================================
1. ARITHMETIC OPERATORS
==================================================

Used for mathematical calculations.

+   Addition
-   Subtraction
*   Multiplication
/   Division
%   Modulus (remainder)
//  Floor Division
**  Power

Example:

a = 10
b = 3

print(a + b)    # 13
print(a - b)    # 7
print(a * b)    # 30
print(a / b)    # 3.3333
print(a % b)    # 1
print(a // b)   # 3
print(a ** b)   # 1000


Explanation:

10 % 3 = 1
Because remainder after division is 1

10 // 3 = 3
Only whole number part

10 ** 3 = 1000
10 × 10 × 10


==================================================
2. COMPARISON OPERATORS
==================================================

Used to compare values.
Result will be True or False.

==   Equal to
!=   Not equal to
>    Greater than
<    Less than
>=   Greater than or equal to
<=   Less than or equal to

Example:

x = 10
y = 20

print(x == y)   # False
print(x != y)   # True
print(x < y)    # True
print(x > y)    # False
print(x <= y)   # True


==================================================
3. LOGICAL OPERATORS
==================================================

Used to combine conditions.

and
or
not

Example:

age = 20

print(age > 18 and age < 30)   # True
print(age < 18 or age > 10)    # True
print(not(age > 18))           # False


Explanation:

and = both conditions must be True
or  = any one condition True
not = reverse result


==================================================
4. ASSIGNMENT OPERATORS
==================================================

=    assign value
+=   add and assign
-=   subtract and assign
*=   multiply and assign
/=   divide and assign

Example:

x = 5
x += 2
print(x)   # 7

x *= 3
print(x)   # 21


==================================================
5. OPERATOR PRECEDENCE
==================================================

Python solves operations like mathematics.

Priority:

1. ()
2. **
3. * / // %
4. + -

Example:

print(2 + 3 * 4)      # 14
print((2 + 3) * 4)    # 20


==================================================
REAL EXAMPLE
==================================================

marks = 85

print(marks >= 35)   # True

DAY: 5
TITLE: Type Casting in Python

EXPLANATION:

Type casting means converting one data type into another data type.

In Python, many times user input comes as STRING, but we may need INTEGER or FLOAT for calculations.

Example:

age = input("Enter age: ")

Even if user types:
19

Python stores it as:

"19"

This is a string, not a number.

So we use type casting.


==================================================
1. COMMON TYPE CASTING FUNCTIONS
==================================================

int()     -> Converts to Integer
float()   -> Converts to Decimal Number
str()     -> Converts to String
bool()    -> Converts to Boolean


==================================================
2. CONVERT STRING TO INTEGER
==================================================

num = "25"
x = int(num)

print(x)
print(type(x))

OUTPUT:
25
<class 'int'>


Useful when taking numeric input:

age = int(input("Enter age: "))


==================================================
3. CONVERT STRING TO FLOAT
==================================================

price = "99.50"
p = float(price)

print(p)
print(type(p))

OUTPUT:
99.5
<class 'float'>


Useful for decimal values.


==================================================
4. CONVERT NUMBER TO STRING
==================================================

age = 20
text = str(age)

print(text)
print(type(text))

OUTPUT:
20
<class 'str'>


Useful when joining text:

age = 20

print("Age is " + str(age))


==================================================
5. CONVERT TO BOOLEAN
==================================================

print(bool(1))        # True
print(bool(0))        # False
print(bool("Hello"))  # True
print(bool(""))       # False


Rules:

0, empty string, empty values = False
Other values = True


==================================================
6. WHY TYPE CASTING IS IMPORTANT
==================================================

Without int():

a = input("Enter first number: ")
b = input("Enter second number: ")

print(a + b)

If user enters:
5
3

OUTPUT:
53

Because strings join together.


With int():

a = int(input("Enter first number: "))
b = int(input("Enter second number: "))

print(a + b)

OUTPUT:
8


==================================================
7. INVALID CONVERSION
==================================================

x = int("hello")

This gives error because text cannot become integer.

Only numeric strings like:

"10"
"25"
"300"

can convert to int.


==================================================
8. MULTIPLE EXAMPLES
==================================================

marks = "85"
height = "5.9"

print(int(marks))
print(float(height))
print(str(500))

DAY: 6
TITLE: Conditional Statements (if, elif, else)

EXPLANATION:

Conditional statements allow a program to make decisions.

A program can check conditions and perform different actions depending on whether the condition is True or False.

Example:

If it is raining:
    take umbrella
Else:
    go normally

In Python, we use:

if
elif
else


==================================================
1. BASIC if STATEMENT
==================================================

Syntax:

if condition:
    statement

Example:

age = 20

if age >= 18:
    print("You are eligible to vote")

OUTPUT:
You are eligible to vote


Explanation:

Since age is 20, the condition age >= 18 is True.


==================================================
2. if WITH FALSE CONDITION
==================================================

age = 15

if age >= 18:
    print("Eligible")

OUTPUT:
(no output)

Because condition is False.


==================================================
3. if ... else
==================================================

Used when there are two possible results.

Syntax:

if condition:
    statement
else:
    statement

Example:

age = 16

if age >= 18:
    print("Eligible to vote")
else:
    print("Not eligible")

OUTPUT:
Not eligible


==================================================
4. if ... elif ... else
==================================================

Used for multiple conditions.

Syntax:

if condition1:
    statement
elif condition2:
    statement
else:
    statement

Example:

marks = 75

if marks >= 90:
    print("Grade A+")
elif marks >= 75:
    print("Grade A")
elif marks >= 50:
    print("Pass")
else:
    print("Fail")

OUTPUT:
Grade A


==================================================
5. INDENTATION IN PYTHON
==================================================

Python uses spaces (indentation) to define blocks of code.

Correct:

if True:
    print("Hello")

Wrong:

if True:
print("Hello")

Indentation is compulsory.


==================================================
6. USING LOGICAL OPERATORS
==================================================

age = 25

if age >= 18 and age <= 30:
    print("Young Adult")

OUTPUT:
Young Adult


==================================================
7. NESTED if
==================================================

age = 22
citizen = True

if age >= 18:
    if citizen:
        print("Can vote")
    else:
        print("Citizen required")

OUTPUT:
Can vote


==================================================
8. REAL-LIFE EXAMPLES
==================================================

Password check
ATM balance check
Login system
Exam result
Weather app


==================================================
9. SIMPLE EVEN / ODD CHECK
==================================================

    print("Even")
else:
    print("Odd")

OUTPUT:
Even

DAY: 7
TITLE: Nested Conditions and Advanced if-else

EXPLANATION:

In Day 6, you learned basic if, elif, else statements.

Today we learn:

1. Nested if statements
2. Multiple condition checking
3. Logical operators with conditions
4. Real-world decision programs

Nested conditions are conditions inside another condition.


==================================================
1. WHAT IS NESTED if?
==================================================

Syntax:

if condition1:
    if condition2:
        statement

Example:

age = 20
has_id = True

if age >= 18:
    if has_id:
        print("Entry Allowed")

OUTPUT:
Entry Allowed


Explanation:

First condition:
age >= 18  → True

Second condition:
has_id → True

So final output prints.


==================================================
2. NESTED if WITH else
==================================================

age = 16
has_id = True

if age >= 18:
    if has_id:
        print("Entry Allowed")
    else:
        print("ID Required")
else:
    print("Age Not Allowed")

OUTPUT:
Age Not Allowed


==================================================
3. USING and OPERATOR
==================================================

Instead of nested if, we can write:

age = 20
has_id = True

if age >= 18 and has_id:
    print("Entry Allowed")

OUTPUT:
Entry Allowed


This is shorter and cleaner.


==================================================
4. USING or OPERATOR
==================================================

rain = True
umbrella = False

if rain or umbrella:
    print("Go Outside Carefully")

OUTPUT:
Go Outside Carefully


If any one condition is True, output runs.


==================================================
5. USING not OPERATOR
==================================================

logged_in = False

if not logged_in:
    print("Please Login")

OUTPUT:
Please Login


not reverses True to False or False to True.


==================================================
6. MULTI-CONDITION GRADE CHECKER
==================================================

marks = 82

if marks >= 90:
    print("A+")
elif marks >= 75 and marks < 90:
    print("A")
elif marks >= 50:
    print("Pass")

    print("Fail")

A


==================================================
7. BIGGEST OF THREE NUMBERS
==================================================

    print("b is biggest")

range(5)

1
2
3
4


Important:
range(5) starts from 0 and stops before 5.

7. SUM OF NUMBERS

DAY: 9
TITLE: while Loop in Python

EXPLANATION:

A while loop is used to repeat a block of code as long as a condition is True.

Unlike a for loop, while loop is useful when we do NOT know exactly how many times to repeat.

Example:

Keep asking password until correct.
Keep running game until user quits.
Count until a condition changes.


==================================================
1. BASIC while LOOP
==================================================

Syntax:

while condition:
    statement


Example:

i = 1

while i <= 5:
    print(i)
    i = i + 1

OUTPUT:
1
2
3
4
5


Explanation:

- Start with i = 1
- Check i <= 5
- Print i
- Increase i by 1
- Repeat until condition becomes False


==================================================
2. IMPORTANCE OF UPDATING VARIABLE
==================================================

i = 1

while i <= 3:
    print(i)
    i = i + 1

If you forget:

i = i + 1

Then loop runs forever (infinite loop).


==================================================
3. INFINITE LOOP
==================================================

while True:
    print("Running")

This keeps running until manually stopped.


==================================================
4. COUNTDOWN PROGRAM
==================================================

n = 5

while n > 0:
    print(n)
    n = n - 1

print("Done!")

OUTPUT:
5
4
3
2
1
Done!


==================================================
5. SUM OF NUMBERS
==================================================

i = 1
total = 0

while i <= 5:
    total = total + i
    i = i + 1

print(total)

OUTPUT:
15


==================================================
6. USER INPUT LOOP
==================================================

password = ""

while password != "python123":
    password = input("Enter password: ")

print("Access Granted")


This repeats until correct password is entered.


==================================================
7. while WITH else
==================================================

i = 1

while i <= 3:
    print(i)
    i += 1
else:
    print("Loop Finished")

OUTPUT:
1
2
3
Loop Finished


==================================================
8. for LOOP vs while LOOP
==================================================

for loop:
Use when repetitions are known.

while loop:
Use when condition controls repetition.

hello

Useful for cleanup work.

EXPLANATION:

1. Inheritance
2. Encapsulation
3. Polymorphism

These are used in professional software development.


==================================================
1. INHERITANCE
==================================================

Inheritance means one class gets features of another class.

Parent class -> base class
Child class -> derived class

Example:

class Animal:
    def sound(self):
        print("Animal makes sound")

class Dog(Animal):
    pass

d = Dog()
d.sound()

OUTPUT:
Animal makes sound


Dog inherited sound() from Animal.


==================================================
2. CHILD CLASS WITH OWN METHOD
==================================================

class Animal:
    def eat(self):
        print("Eating")

class Dog(Animal):
    def bark(self):
        print("Barking")

d = Dog()

d.eat()
d.bark()

OUTPUT:
Eating
Barking


==================================================
3. METHOD OVERRIDING
==================================================

Child class can replace parent method.

class Animal:
    def sound(self):
        print("Animal sound")

class Dog(Animal):
    def sound(self):
        print("Bark")

d = Dog()
d.sound()

OUTPUT:
Bark


==================================================
4. super() FUNCTION
==================================================

Used to call parent class methods.

class Animal:
    def __init__(self):
        print("Animal created")

class Dog(Animal):
    def __init__(self):
        super().__init__()
        print("Dog created")

d = Dog()

OUTPUT:
Animal created
Dog created


==================================================
5. ENCAPSULATION
==================================================

Encapsulation means protecting data inside class.

Use underscore naming:

class Bank:

    def __init__(self):
        self._balance = 1000

b = Bank()
print(b._balance)

_single underscore means internal use.


==================================================
6. PRIVATE VARIABLE
==================================================

class Bank:

    def __init__(self):
        self.__pin = 1234

Double underscore makes name private-like.

Cannot access directly easily:

b = Bank()
# b.__pin  -> Error


==================================================
7. GETTER METHOD
==================================================

class Bank:

    def __init__(self):
        self.__balance = 5000

    def get_balance(self):
        return self.__balance

b = Bank()
print(b.get_balance())

OUTPUT:
5000


==================================================
8. POLYMORPHISM
==================================================

Same method name, different behavior.

class Cat:
    def sound(self):
        print("Meow")

class Dog:
    def sound(self):
        print("Bark")

for animal in [Cat(), Dog()]:
    animal.sound()

OUTPUT:
Meow
Bark


==================================================
9. WHY THESE CONCEPTS?
==================================================

Inheritance:
Reuse code

Encapsulation:
Protect data

Polymorphism:
Flexible design

students = {}

def add_student():
    name = input("Enter name: ")
    marks = int(input("Enter marks: "))
    students[name] = marks
    print("Student added successfully")

def view_students():
    if not students:
        print("No students found")
    else:
        for name, marks in students.items():

    name = input("Enter name to search: ")
    if name in students:
        print(name, "Marks:", students[name])
    else:
        print("Student not found")

        break
